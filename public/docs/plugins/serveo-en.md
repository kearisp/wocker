# Serveo

The plugin runs [serveo.net](https://serveo.net), with its help you can create a tunnel for the project.


## Installation

```shell
ws plugin:add serveo
```


## Commands

```shell
ws serveo:init
ws serveo:start
ws serveo:stop
ws serveo:logs
```


## Setup up

```shell
cd /project-dir
```

```shell
ws serveo:init
```

[starting project](/project/management#starting-project).
