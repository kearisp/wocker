# Логування

```shell
ws logs [options]
```

Команда `ws logs` використовується для перегляду логів проєкту або глобальних логів `wocker`.

### Опції

- `-n, --name <name>` — Ім'я проєкту. Дозволяє переглядати логи конкретного проєкту з будь-якої директорії.
- `-f, --follow` — Продовжувати виводити логи в реальному часі (stream).
- `-g, --global` — Виводити глобальні логи `wocker` (системні повідомлення CLI).
- `-c, --clear` — Очистити файл логів. Працює лише разом із опцією `--global`.
- `-d, --detach` — Запустити перегляд логів у фоновому режимі (стосується контейнерів проєкту).

### Приклади

Перегляд логів поточного проєкту:
```shell
ws logs
```

Перегляд глобальних логів `wocker` у режимі реального часу:
```shell
ws logs -g -f
```

Очищення глобальних логів:
```shell
ws logs -g -c
```
