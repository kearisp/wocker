# Project Management


## Starting project

From project directory

```shell
ws start
```

From another directory

```shell
ws start --name=project-name
```


## Stopping project

From project directory

```shell
ws stop
```

From another directory

```shell
ws stop --name=project-name
```

## Logging

To view project logs, use the `ws logs` command.

```shell
ws logs
```

Read more in the [Logging](/docs/project/logs) section.
