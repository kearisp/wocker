# Cron


## Installation

```shell
ws plugin:add cron
```


# Commands

```shell
ws cron:start
ws cron:stop
ws crontab -e
ws crontab -l
ws crontab -r
```


## Starting cron service

The crontab of running projects will be automatically added to the crontab service.

```shell
ws cron:start
```


## Stop cron

```shell
ws cron:stop
```
